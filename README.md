# 📚 복습플랜 (ReviewPlan)

고등학생을 위한 **공부시간 체크 + 복습 진도 관리** 웹앱.
학교에서 배운 내용을 입력하면, **시험 전까지 적절한 횟수로 복습**할 수 있도록
망각곡선 기반 간격 반복(spaced repetition) 일정을 자동으로 만들어 줍니다.

설치·서버·로그인 없이 **브라우저로 바로 사용**하며, 데이터는 기기에 저장됩니다(localStorage).
휴대폰 브라우저에서도 잘 작동합니다.

## ✨ 주요 기능

- **🔁 복습 진도 관리**
  - 과목 / 단원(배운 내용) / 수업한 날 / 시험 날을 입력하면 복습 일정 자동 생성
  - 간격 반복 알고리즘(1·3·7·14·30일 + 시험 직전 최종 점검)으로 시험까지 기간에 맞춰 횟수 조절
  - 복습할 때마다 체크 → 진행률·밀린 복습을 한눈에 확인
- **⏱️ 공부시간 체크**
  - 과목별 스톱워치 타이머로 공부시간 측정 후 저장
  - 시간을 직접 입력해 기록 추가도 가능
- **🏠 홈 대시보드**
  - 오늘 복습할 항목, 오늘 공부시간, 다가오는 시험(D-day)
- **📊 통계**
  - 최근 7일 공부시간 그래프, 과목별 누적 시간, 복습 진행률

## 🚀 사용법

저장소를 받은 뒤 `index.html`을 브라우저로 열기만 하면 됩니다.

```bash
# 또는 간단한 로컬 서버로 실행
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 🧠 복습 일정은 어떻게 정해지나요?

수업일과 시험일 사이 기간을 기준으로, 망각곡선에서 자주 쓰이는
간격(1, 3, 7, 14, 30일)을 적용해 복습 날짜를 배치합니다.

- 시험까지 기간이 길면 → 복습 횟수가 늘어납니다.
- 기간이 짧으면 → 핵심 시점만 압축해서 배치합니다.
- 시험 직전(전날 또는 당일)에는 항상 **최종 점검 복습**을 넣습니다.

## 🗂️ 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 |
| `style.css` | 모바일 우선 스타일 |
| `app.js` | 복습 일정 생성·타이머·통계 로직 (localStorage) |

## 🔒 데이터

모든 데이터는 사용 중인 브라우저에만 저장되며 외부로 전송되지 않습니다.
‘통계’ 탭의 **모든 데이터 초기화**로 언제든 지울 수 있습니다.
